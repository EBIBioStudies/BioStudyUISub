1.Starting a new submission

2.Creating a Biostudy

Use the toolbar on left side of screen for completing information about the BioStudy.

Please note that submissions normally require a title, a release date for the study, and a description.

The BioStudy will remain private and will only be accessed using login credentials until the release date.

7.Uploading files

All files must be uploaded first, before being used in a BioStudy. The files for each user are stored in a central repository and are available for all submissions.

To load files select from the toolbar. This displays all the files that have been uploaded by the user.

9.Saving and submitting a BioStudy

Studies are automatically saved, so you can leave an incomplete submission and return to it later.

Press "Submit" to receive an accession number and load the submission into BioStudies.
