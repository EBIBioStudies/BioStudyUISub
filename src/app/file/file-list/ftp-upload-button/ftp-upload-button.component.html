The Aspera ascp command line client is distributed as part of the ‘Aspera Connect’ high-performance transfer browser plug-in which can be downloaded from http://www.asperasoft.com/downloads/connect. Your command for uploading a directory can be:

    ascp -P33001 -i <path to asperaweb_id_dsa.openssh file> -d <local directory to upload> bsaspera_w@hx-fasp-1.ebi.ac.uk:.dropbox/<secret directory>
where
  • -P33001 and bsaspera@fasp-beta.ebi.ac.uk defines port, user and server for Aspera connection
  • -d tells the server to create the directory if it does not exist
  • asperaweb_id_dsa.openssh is the public key for Aspera connection. <path to asperaweb_id_dsa.openssh file> might be:
    1. on Linux: <aspera connect installation directory>/etc/asperaweb_id_dsa.openssh
    2. on Mac OSX: <aspera connect installation directory>/asperaweb_id_dsa.openssh
    3. on Windows: "%userprofile%\AppData\Local\Programs\Aspera\Aspera Connect\etc\asperaweb_id_dsa.openssh"
  • <secret directory> looks like xx/xxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx-xxxx
